# 🚀 Guia de UI/UX com IA integrada

Projeto desenvolvido com foco em aprendizado de UI/UX aliado à integração com Inteligência Artificial.

## 🧠 Sobre o projeto

Este é um website educacional que apresenta conceitos de UI/UX e permite interação com um assistente de IA para:

- Geração de resumos

- Explicações de conteúdo

- Apoio ao estudo

## 🛠️ Tecnologias utilizadas

- HTML, CSS, JavaScript

- Integração com API de IA

## 🤖 Funcionalidades

- Interface responsiva

- Modo claro/escuro

- Navegação por conteúdos
